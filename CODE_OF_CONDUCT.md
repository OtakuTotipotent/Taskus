# Contributor Covenant Code of Conduct (short)

We follow the Contributor Covenant. In short:

- Be respectful and kind.
- No harassment, discrimination, or abusive language.
- Be collaborative and constructive in reviews.

If you experience unacceptable behavior, contact the repo maintainers: maintainers@your-domain-or-github-username.

Full text: <https://www.contributor-covenant.org/version/2/0/code_of_conduct/>
